# Xサーチバレット

X（旧Twitter）の検索をワンクリックで実行・コマンドをコピーできる Chrome 拡張機能です。

## 機能

- **X を直接開く** — ボタン1つで検索結果ページを新しいタブで開く
- **クリップボードにコピー** — コマンドをコピーして任意の場所に貼り付け
- **複数ユーザーID管理** — よく使うアカウントを登録・切り替えして `from:` 系コマンドに自動反映
- **until: 自動補正** — X の仕様（指定日を含まない）を考慮して終了日を自動で +1日補正
- **OR 検索** — スペース区切りで入力すると `(AI OR Gemini OR Claude)` に自動変換
- **除外ワード** — スペース区切りで入力すると `-求人 -採用 -PR` に自動変換

## コマンドカテゴリ

| カテゴリ | 内容 |
|---|---|
| よく使うおすすめ（返信・リポスト除く） | 返信・リポストを除いた実用テンプレ |
| 選択IDのポスト | 選択IDの投稿を期間・いいね数などで検索 |
| キーワード検索 | OR検索・除外ワード・期間・いいね数など |
| フィルター | 返信・リポスト・画像・動画のフィルタリング |

## インストール方法

Chrome ウェブストアには未公開のため、デベロッパーモードで読み込んでください。

1. Chrome で `chrome://extensions/` を開く
2. 右上の **「デベロッパーモード」** をオンにする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. このリポジトリのフォルダを選択

## 使い方

1. ブラウザ右上の拡張機能アイコンをクリック
2. **ユーザーID バー** をクリックして登録パネルを開く
3. 表示名と @username を入力して「追加」
4. 複数登録している場合はラジオボタンで使用するIDを切り替え
5. 使いたいコマンドの入力欄を埋める
6. **「開く」** で X の検索結果を直接開く、または **「📋」** でコマンドをコピー

## 注意事項

- 動作確認済みですが、Xの仕様変更により予告なく動かなくなる場合があります
- X の検索コマンド（`from:`, `since:`, `min_faves:` など）は公式にドキュメント化されていない非公式仕様です
- `until:` は当日を含まない X の仕様のため、本ツールでは終了日を自動で +1日して送信しています
- 条件が厳しいと、X側「話題のポスト」「最新」タブのいずれかに結果が出ないことがあります。その場合はいいね数の条件を下げるか期間を広げてみてください
- 日付範囲が狭すぎると一部の投稿が出ない場合があります。その場合は範囲を広げてみてください（Xの検索インデックスの仕様によるもので、ツール側では制御できません）
